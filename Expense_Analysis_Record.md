# Power BI 企業費用分析 V2 儀表板 — 修改紀錄

## 版本：V2.0 | 日期：2026-03-07

---

## 一、現有 V1 分析結論

### 費用分析核心重點（依教材萃取）

| 分析維度 | 說明 | DAX 對應 |
|---------|------|---------|
| 費用結構分析 | 各科目佔比，識別主要費用來源 | `Pct_of_Total` |
| 費用趨勢追蹤 | YoY/MoM 同環比變化 | `YoY_Pct`, `MoM_Pct` |
| 預算達成率分析 | 實際 vs 預算差異、超支判斷 | `Achieve_Rate`, `Variance` |
| 部門/專案貢獻 | 費用分攤至部門/專案 | `Rank_Dept` |
| 異常偵測 | 超預算 15%+ 或 MoM>30% | `Overspend_Level` |

### V1 主要問題
- 範例資料僅19筆，趨勢分析意義有限
- DAX 公式頁只有4組，遠不完整
- 缺少情境模擬(What-If)功能
- 缺少成本效益指標(CAGR、人均費用)
- 部分圖表（Donut、Polar）渲染不穩定

## 二、完整 DAX 公式彙整（V2 共 12 組）

| # | 公式名稱 | 說明 |
|---|---------|------|
| ① | `Total_Expense` | 實際費用加總（所有計算基礎） |
| ② | `Budget_Expense` | 預算額度加總 |
| ③ | `Budget_Remain` | 尚可動用的預算空間 |
| ④ | `Variance` | 差異值（正值=超支） |
| ⑤ | `Achieve_Rate` | 費用達成率（>100%=超支） |
| ⑥ | `Expense_YTD` | 年初至今累計費用（時間智慧） |
| ⑦ | `YoY_Pct` | 年度同比增長率 |
| ⑧ | `MoM_Pct` | 月環比增長率 |
| ⑨ | `Pct_of_Total` | 各類別費用佔總費用比例 |
| ⑩ | `Overspend_Level` | 超支三級自動分類（嚴重/警告/輕微） |
| ⑪ | `Rolling3M/12M` | 3/12個月滾動均值（消除季節性） |
| ⑫ | `Per_Head` / `Expense_CAGR` | 人均費用效率 / 年化複合增長率 |

## 四、V2 新增功能說明

### 🔮 What-If 情境模擬（P8）
- 6個費用科目各有 -30% ~ +30% 滑桿
- 即時計算模擬後總費用、預算達成率、預算餘裕
- 視覺化對比（實際/模擬/預算三柱圖）

### 🚨 異常偵測三級分類（P7）
- 🔴 **嚴重**：超預算 > 25%
- 🟡 **警告**：超預算 10~25%
- 🟢 **輕微**：超預算 0~10%

### 📥 資料匯入引擎
- 支援 `.csv` / `.xlsx` / `.xls`
- CSV：FileReader + 手動解析標頭
- Excel：SheetJS XLSX 函式庫解析
- 新欄位：`Region`（區域）、`Employee_Count`（人員數）

## 五、修改建議摘要

1. **增加瀑布圖**：已實作，視覺化各科目對整體預算差異的貢獻
2. **增加情境模擬**：已實作 What-If 滑桿，6科目調整
3. **改善異常偵測**：已升級為三級自動評分系統
4. **增強範例資料**：從19筆擴充為44筆，涵蓋12個月
5. **擴充DAX頁**：從4組擴充至12組完整公式
6. **增加月度堆疊圖**：追蹤費用科目結構隨月份的變化
7. **新增雷達圖**：費用科目效益多維比較
8. **新增進度條**：各科目預算消耗直觀呈現

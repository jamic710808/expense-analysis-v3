    🎯 費用科目效益雷達

    💡 Power BI 費用分析 DAX 完整公式參考庫

    ① 基礎度量值

    Total_Expense = SUM(Expense[Actual_Amount])
Budget_Expense = SUM(Expense[Budget_Amount])
Budget_Remain = [Budget_Expense]-[Total_Expense]

    所有計算的基礎。Budget_Remain為剩餘預算空間。

    ② 差異與達成率

    Variance = [Total_Expense] - [Budget_Expense]
Achieve_Rate = DIVIDE([Total_Expense],[Budget_Expense],0)
Overspend_Pct = [Achieve_Rate] - 1

    Achieve_Rate > 1 = 超支；Overspend_Pct為超支百分比。

    ③ 時間智慧 YTD/YoY

    Expense_YTD =
  TOTALYTD([Total_Expense],'Date'[Date])
Expense_LY =
  CALCULATE([Total_Expense],
    SAMEPERIODLASTYEAR('Date'[Date]))
YoY_Pct = DIVIDE([Total_Expense]-[Expense_LY],
                 [Expense_LY],0)

    YTD累計與年度同比增長率計算。

    ④ 環比 MoM

    Expense_LM =
  CALCULATE([Total_Expense],
    DATEADD('Date'[Date],-1,MONTH))
MoM_Pct = DIVIDE([Total_Expense]-[Expense_LM],
                 [Expense_LM],0)
MoM_Var = [Total_Expense] - [Expense_LM]

    偵測月度費用突發性異常波動。

    ⑤ 排名與佔比

    Pct_of_Total =
  DIVIDE([Total_Expense],
    CALCULATE([Total_Expense],
      ALL(Expense_Category)),0)
Rank_Dept =
  RANKX(ALL(Department),[Total_Expense],,DESC)

    找出費用集中度最高的科目與部門。

    ⑥ 超支分級

    Overspend_Level =
  VAR R = [Achieve_Rate]
  RETURN
  IF(R>1.25,"🔴 嚴重",
  IF(R>1.10,"🟡 警告",
  IF(R>1.00,"🟢 輕微","✅ 正常")))

    自動三級超支預警，對應異常偵測頁面。

    ⑦ 滾動均值

    Rolling3M =
  AVERAGEX(DATESINPERIOD('Date'[Date],
    LASTDATE('Date'[Date]),-3,MONTH),
    [Total_Expense])
Rolling12M =
  AVERAGEX(DATESINPERIOD('Date'[Date],
    LASTDATE('Date'[Date]),-12,MONTH),
    [Total_Expense])

    消除季節性，觀察費用長期趨勢。

    ⑧ 人均費用

    Per_Head =
  DIVIDE([Total_Expense],
    SUM(Expense[Employee_Count]),0)
Dept_Efficiency =
  DIVIDE([Per_Head],
    CALCULATE([Per_Head],ALL(Department)),0)

    評估各部門人力資源使用效率指標。

    ⑨ 費用集中度

    Concentration =
  VAR MaxCat = CALCULATE(MAX([Total_Expense]),
    ALL(Expense_Category))
  RETURN DIVIDE(MaxCat,
    CALCULATE([Total_Expense],
      ALL(Expense_Category)),0)

    最大費用科目的集中度，>50%需關注。

    ⑩ 前10大帕累托

    Top10_Pct =
  VAR T = CALCULATE([Total_Expense],
    TOPN(10,ALL(Expense_Category),
         [Total_Expense],DESC))
  VAR A = CALCULATE([Total_Expense],
    ALL(Expense_Category))
  RETURN DIVIDE(T,A,0)

    80/20費用集中分析，支援資源優化決策。

    ⑪ 動態What-If參數

    Simulated_Expense =
  [Total_Expense] * (1 + SELECTEDVALUE(
    Params[AdjustPct],0))
Dynamic_Metric =
  SWITCH(SELECTEDVALUE(Params[Metric]),
    "費用",[Total_Expense],
    "預算",[Budget_Expense],
    "差異",[Variance])

    情境模擬必須，支援What-If分析頁面。

    ⑫ CAGR複合增長率

    Expense_CAGR =
  VAR S = CALCULATE([Total_Expense],
    FIRSTDATE('Date'[Date]))
  VAR E = CALCULATE([Total_Expense],
    LASTDATE('Date'[Date]))
  VAR N = DATEDIFF(
    FIRSTDATE('Date'[Date]),
    LASTDATE('Date'[Date]),YEAR)
  RETURN POWER(DIVIDE(E,S,0),
    DIVIDE(1,MAX(N,1),0))-1

    費用的年化複合增長率，用於長期規劃。
